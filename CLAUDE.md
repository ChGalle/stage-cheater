## Project Overview

Stage-Cheater is a Raspberry Pi-based stage teleprompter for displaying song lyrics and chords. The system displays ChordPro files, controlled via foot pedal, with playlist support for organizing songs.

## Key Features (from PROJECT.md)

- ChordPro file format support for lyrics/chords display
- Foot pedal control (Bluetooth keyboard mode OR GPIO pins)
- Adjustable zoom factor
- Playlist files for song ordering
- Auto-mount USB stick for data loading (ChordPro files, playlists, config)
- GPIO pin support for clean shutdown/restart
- Configuration via config file on USB stick
